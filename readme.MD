### APIs -
`POST` to `/postgres-backup` starts a Postgres Database Backup to S3.


### Start the server
```shell
go run server.go
```

### Folders
* /app - application
* /app/backup - backup files

### Environment variables

Database
* DB_USER
* DB_DATABASE
* DB_HOST
* DB_PWD
* DB_PORT
* DB_SSL

S3 API
* S3_ACCESS_KEY
* S3_SECRET
* S3_BUCKET
* S3_REGION
* S3_URL
